# Your own personal library

Check it out live: https://shriyashzzz.github.io/Book-Library/

This project was made in order to practice implementing objects that link to dom and to be able manipulate the nodes in the node using javascript.

Made using HTML, CSS ,and JACASCRIPT

Recent Update:

Form used to add books are validated on the client-side, helpful info is given when user fills it.

Your own browser local-storage is used to add the books you add to save locally.
Again it's locally saved, not into cloud (cannot be retrieved if localstorage is cleaned)

Anywhoo it's no longer a static site, so you can actually use it if you wish~ yepeee

To Install

Clone this repo:

```bash
    git@github.com:Shriyashzzz/Book-Library.git

```

Install Depenedecies

```bash
    npm install
```

Start developement Server

```bash
npm start

```

This will launch webpack-dev-server and open the app in your browser at http://localhost:8080.

Building for production

```bash
npm run build
```
